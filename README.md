# GovTrack: Government Entity Inventory & Procurement System

## Overview
GovTrack is a comprehensive management system designed for Government-Owned Entities (GOEs) and public sector departments. It streamlines the lifecycle of government property management, ensuring compliance, transparency, and efficiency in procurement and asset tracking.

## Core Features
*   **Intelligent Inventory Management:** Track consumables and assets with automated SKU and Government ID generation.
*   **Procurement Workflow:** A multi-step, role-validated workflow following strict public procurement standards (GSRF, PR, PO, GRN).
*   **Vendor Management:** Maintain a database of approved vendors and a dedicated portal for electronic quote submissions.
*   **Asset Registry:** Manage fixed assets with QR code integration for field audits and lifecycle tracking.
*   **Reporting & Analytics:** Real-time KPIs and interactive charts for inventory distribution and spending trends.
*   **Governance & RBAC:** Role-based access control for Users, Supervisors, Managers, Finance Officers, Stock Keepers, and Admins.
*   **Document Generation:** Automated generation of official documents (POs, GSRFs, Completion Certificates) with customizable templates.
